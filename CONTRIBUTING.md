# Contributing

When contributing to this repository, please first discuss the change you wish to make via issue,
email, or any other method with the owners of this repository before making a change.

Please note we have a [code of conduct](CODE_OF_CONDUCT.md), please follow it in all your interactions with the project.

## Pull Request Checklist

* If you are including functional changes, add tests to the new code
* Run `rake` before raising a PR to lint your code and run the tests
* Ensure your commits follow the Conventional Commits style https://www.conventionalcommits.org/en/v1.0.0/, and if necessary squash any unneeded commits
